Reduce y Filter

Reduce

Reduce es un método que nos permite reducir un array a un solo valor. Este método recibe dos parámetros, el primero es una función que se encargará de realizar la operación y el segundo es el valor inicial.

Filter

Filter es un método que nos permite filtrar los elementos de un array que cumplan con una condición. Este método recibe como parámetro una función que se encargará de realizar la condición.
